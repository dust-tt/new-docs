## Agent management

Agents can be managed through the **Manage** page, accessible from the main page


### Browsing and searching

From there, all accessible Agents can be found easily, via the Search bar, or by filters

- Editable by me : All agents that you can edit - this includes your own Agents and the ones your colleagues added you to as **Editor**
- Default : The Default Agents provided by Dust


### Admins: become an editor of any agent

As a workspace admin, you can open any agent in the Agent Builder, even if you are not one of its editors. In that case, the builder opens in view-only mode with a banner indicating that you are not an editor of the agent.

To make changes, click the **Become an editor** button, available both in the banner and in the **Editors & Access** section of the agent's settings. This adds you to the agent's editors list, after which you can edit and save the agent like any other editor.

The same applies to skills: admins can view any skill in the Skill Builder and click **Become an editor** to add themselves as editors before saving changes.


## Tags

Tags help organize agents in your workspace and make them easier to find. Think of them as **labels** that describe what an agent does, which team uses it, or its purpose.

As an administrator, you play a key role in organizing your workspace's agents through tags. Here's how to set up and manage an effective tagging system.

### Setting Up Tags

Navigate to **Manage Agents** > **Tags** > **Manage Tags**


Only administrators can access the **Managing tags** dashboard and create new tags. In this dashboard you will also be able to edit and delete tags, and check how many agents are tagged.


<Info>
**</Callout**

**Tag suggestions**
If you don't have any tag set up yet, Dust will suggest a list of tags that can be assigned to your existing agents.
</Info>


### Assigning Tags to agents

Once Tags have been created, they can be assigned to existing agents from the agent list to keep things tidy :broom:
