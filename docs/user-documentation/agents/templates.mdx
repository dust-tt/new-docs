---
title: "Templates"
---

Templates help you discover common use cases. Selecting a template opens a Sidekick-guided creation flow pre-loaded with the template's context. Sidekick drafts the initial instructions and suggests tools and data sources; you review, adjust, and publish a customized agent.

### Start from a template

### Follow the template instructions and guidance

Instead of applying static instructions, Sidekick collaborates with you to refine the agent's instructions and proposes relevant tools. Accept, edit, or decline its suggestions, then publish.
