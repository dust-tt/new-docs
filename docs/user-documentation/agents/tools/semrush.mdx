<Callout type="info"**Remote MCP Server**>
</Callout>
The Semrush tool is a **remote MCP** (Model Context Protocol) server hosted and maintained by Semrush. Dust does not own, manage, or provide support for this integration. For any questions about the tools, data availability, or API access, contact Semrush directly or refer to the [Semrush MCP documentation](https://www.semrush.com/kb/1618-mcp).

The Semrush MCP connects Dust agents to live Semrush data via the [Model Context Protocol](https://modelcontextprotocol.io/introduction), an open standard that allows AI tools to query external data sources in real time.

Once connected, agents can access SEO metrics, keyword rankings, competitive intelligence, traffic breakdowns, and backlink data directly from Semrush, without exporting files or switching platforms.

Navigate to **Spaces > Tools** in your Dust workspace, click **Add Tools**, and select **Semrush MCP**.

The setup connects to the Semrush remote MCP server. You will need a valid **Semrush API key** or go through an **OAuth flow** depending on your Semrush plan. Once connected, the tool is available to add to any agent in your workspace.

<Callout type="warning"**Note:** MCP access is available on Semrush One, SEO, and Trends API plans. Check your [Semrush subscription settings](https://www.semrush.com/accounts/subscription-info/api-units/) to confirm you have the required plan and API units.>
</Callout>
In the **Agent Builder**, click **Add Tool** and select **Semrush MCP**.

Agents with this tool can then query Semrush data in natural language. The agent will determine which Semrush report to call based on the user's request.

For the full and up-to-date list of tools and their capabilities, refer directly to the [Semrush MCP documentation](https://www.semrush.com/kb/1618-mcp).

</Callout>
