# How do Customer Support Teams use Dust?

Dust empowers customer support teams with AI-driven tools to streamline issue resolution and enhance customer satisfaction. Dust's versatile features help support teams work more efficiently and effectively.

| Use Case                                   | Task to tackle                                                                                                                                 |
| ------------------------------------------ | ---------------------------------------------------------------------------------------------------------------------------------------------- |
| Support Issue Troubleshooting & Resolution | Assist support teams in troubleshooting customer inquiries with instant internal context from complex FAQs, and providing a tailored response. |
| Customer Feedback Broadcast                | Surface user feedback & customer requests gathered in various places, to highlight recurring customer needs you can take action on.            |
| Support Issue Escalation                   | Enable support team members to escalate issues with the right context, format and routing.                                                     |
| Support Coaching                           | Offer feedback to support team members using real-time best practices and analyzing their ticket history.                                      |
| FAQ Builder                                | Takes resolved tickets and transforms into a FAQ.                                                                                              |
