## 1. Internal Information Retrieval

Instantly access or summarize any document in your knowledge bases, even the ones you are not even aware of.

This is the core function of the **@dust** agent. It is built-in by default in your workspace. You can ask very basic information questions, but also any other question involving reasoning and internal company information.

## 2. Generic Web Search Engine

Access an AI-powered answer engine that searches the internet and provides accurate and real-time answers to any question.\
(Note: when calling directly the LLMs (ex: Claude 4 Sonnet) in the Dust interface, they also have access to the internet)

Follow this [User Guide](https://dust-tt.notion.site/Generic-Web-Search-Engine-a32b786bd4d840639f20a689716f40f9?pvs=4)
