---
title: "What data sources can be connected to Dust?"
---


Dust can plug to a variety of data sources, including native connections, websites, or manual file uploads. You can find all about Data sources at this [Overview](/docs/user-documentation/data-sources/overview).

Especially:

- Dust offers **native connections** with multiple platforms including: Confluence, Google Drive, GitHub, Notion, Slack, Snowflake, Zendesk, Intercom, Microsoft SharePoint. You can find all about connections at [Connections](/docs/user-documentation/data-sources/connections)
- You can also ingest any kind of data leveraging Zapier [\[Zapier\] Automatically add any datasource to Dust](/docs/user-documentation/data-sources/custom-connections/zapier-automatically-add-datasource) or by leveraging our API: [Create a file upload URL](/docs/developer-platform/dust-api-documentation/openapi-and-postman).
