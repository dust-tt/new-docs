Dust can plug to a variety of data sources, including native connections, websites, or manual file uploads. You can find all about Data sources at this [Overview](././what-are-data-sources).

Especially:

* Dust offers **native connections** with multiple platforms including: Confluence, Google Drive, GitHub, Notion, Slack, Snowflake, Zendesk, Intercom, Microsoft SharePoint. You can find all about connections at [Connections](././connections)  
* You can also ingest any kind of data leveraging Zapier [\[Zapier\] Automatically add any datasource to Dust](doc:how-to-automatically-add-data-sources-to-dust-folders-with-zapier) or by leveraging our API: [Create a file upload URL](./ref:post_api-v1-w-wid-files).
