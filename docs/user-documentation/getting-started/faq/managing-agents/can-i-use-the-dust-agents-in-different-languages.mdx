---
title: "Can I use the Dust agents in different languages?"
---


Absolutely! Dust agents have strong multilingual capabilities. Here's what you need to know:

### How it works

Dust agents are powered by large language models (LLMs) that natively understand and generate text in a wide variety of languages. This means you can:

- **Write to an agent in any language** and it will understand you
- **Receive responses in your language** - agents naturally reply in the language you use
- **Set instructions in one language** while interacting in another - the agent adapts to the conversation language

### Key highlights

- **50+ languages** supported for drafting messages
- **No special configuration needed** for most use cases - just write in your preferred language
- **Voice mode** also supports transcription across **70+ languages** with automatic detection

### Tips for your agents

- If you want an agent to **always respond in a specific language** regardless of the input language, you can specify that in the agent's instructions (e.g. "Always respond in French")
- If you want the agent to **match the user's language**, you can instruct it to do so explicitly, or simply leave it without language instructions (it will naturally mirror the user's language)
- **Data sources connected to your agents are language-agnostic** - the agent will retrieve and summarize content even if it's in a different language than the question
