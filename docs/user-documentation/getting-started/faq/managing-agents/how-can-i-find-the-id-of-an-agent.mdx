In the list of your agents, click on the "**...**" then "**More info**"

<Image align="center" src="https://files.readme.io/f8ef031dbebad67bae286513b64bd5fb608ea054e33b73df11ae8057f7b75d84-Screenshot_2024-10-01_at_16.39.36.png" />

A side panel will open. Click again on the "...", you will then see a button to copy the agent ID to your clipboard
