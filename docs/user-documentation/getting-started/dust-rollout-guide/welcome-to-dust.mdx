## Overview of Dust

Dust is a next-generation enterprise AI platform that transforms how organizations leverage artificial intelligence by seamlessly connecting internal knowledge, tools, and workflows. Unlike traditional AI solutions that operate in isolation, Dust lets teams create contextualized AI agents that understand and interact with your company's data and systems.

## Core value proposition

**1. Unified AI-powered workspace**

- Seamlessly integrates with existing enterprise tools and data sources (Slack, Google Drive, Notion, Confluence, GitHub, etc.)
- Creates a centralized hub for AI-assisted work while maintaining data security and governance.
- Enables natural language interaction with company knowledge and systems.

**2. Builder-centric platform**

- Empowers a community of internal builders to create custom AI agents.
- Provides a gallery for discovery and Sidekick‑guided creation for quick deployment.
- Supports both technical and non-technical users in creating AI workflows.

**3. Enterprise-grade security & control**

- Maintains strict data privacy with no training on company data.
- Offers comprehensive admin controls and usage analytics.
- Supports SSO integration, access control policy and role-based access management.

**4. Advanced technical capabilities**

- Retrieval Augmented Generation (RAG) for accurate, context-aware responses.
- Quantitative analysis tools, like Table Query, to combine semantic search capabilities with SQL to analyze data.
- Multi-modal capabilities, including image analysis and data visualization.
- Extensible platform through Dust Apps for custom integration and LLM orchestration.

## Key differentiators

**1. Unmatched flexibility & integration**

- Model-agnostic approach ensures access to the best AI models for each task and avoids overfitting risks.
- Seamless integration with existing enterprise tools and workflows via native Connections or the Dust API.
- Chrome extension and API access to bring Dust agents right in the tools you work in.

**2. Enhanced capability through context**

- Deep integration with company knowledge bases and data sources.
- Real-time synchronization ensures agents always work with current information.
- Advanced tools for complex tasks (transcripts/feedback analysis, visualization, table queries).
- Ability to handle structured and unstructured data across multiple sources.

**3. Customization without compromise**

- No-code agents creation and customization in minutes.
- Granular control over instructions, data sources, and tool selection.
- Builder community to access knowledge sharing and best practices.
- Continuous improvement through user feedback loops.
- White-glove success and onboarding support.

**4. Enterprise-grade extensibility**

- Developer platform for creating custom apps and integrations.
- API-first architecture enables deep workflow automation.

**5. Measurable business impact**

- Proven ROI through automated workflows and time savings.
- Reduced dependency on multiple-point solutions.
- Accelerated decision-making through contextual and advanced intelligence.
- Scalable approach to AI implementation across teams.
