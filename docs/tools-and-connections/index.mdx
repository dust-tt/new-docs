# Tools & Connections

Tools enable your agents to take actions and retrieve information from external services. Connections synchronize data from your tools into Dust. MCP servers let you extend Dust's toolset beyond the default list.

- **[Tools management](./tools-management)** — view, add, remove and configure tools
- **[Connections Management](./connections-management)** — add and manage data connections
- **[Adding an MCP Server](./adding-mcp-server)** — extend Dust's toolset with remote MCP servers
- **[Personal vs Shared Credentials](./personal-vs-workspace-credentials)** — choose the right authentication mode
- **[Agents management](./agents-management)** — browse, search, and tag agents
